What is it?
-----------
The processor.py is the Python solution for extracting and reformatting compressed xml file.


Dev
------------
Python 2.7.6 on Ubuntu x64
Dependencies are listed in requirements.txt


Start
------------
[sudo -H] pip install -r requirements.txt


Usage
------------
usage: processor.py [-h] [--show] [--save] input_file [input_file ...]


Tests
------------
python processor_test.py
